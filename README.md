# Awesome Prompts

> 🧠 我的常用高质量提示词仓库
> 我会持续把自己在工作 / 学习 / 创作中反复验证“好用、稳定、可复用”的提示词整理在这里。

---

## ✨ 仓库包含什么？

- ✅ 可直接复制使用的 Prompt（提示词）
- ✅ 每个 Prompt 都会有：用途 / 用法 / 注意事项 / 示例
- ✅ 适配常见场景：内容生成、写作润色、汇报总结、研究分析、AI 工具使用、效率提升等
- ✅ 尽量做到“拿来就能用”，同时也便于你二次改造

---

## 🚀 如何使用？

1. 在下方 **Prompt 索引** 中找到你需要的提示词
2. 点击 Prompt 名称跳转到 README 对应章节查看说明
3. 或点击「文件」直接打开 `.prompt.md` 原始 Prompt
4. 复制 Prompt → 替换你的变量 → 使用
